# User Catalog

Каталог пользователей на React + TypeScript с поиском и пагинацией.
Данные: [dummyjson.com/docs/users](https://dummyjson.com/docs/users)

## Запуск

**Требования:** Node.js 18+

```bash
# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```

Откроется по адресу: http://localhost:5173

## Сборка для production

```bash
npm run build
npm run preview
```

## Стек

- React 19
- TypeScript
- Vite
- CSS Modules
